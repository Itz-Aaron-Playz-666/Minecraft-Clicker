# Security Policy

## Supported Versions
We provide security updates for the all versions of this project

## Reporting a Vulnerability
If you discover a security issue in this project:

1. **Do not open a public issue.**
2. Instead, please email the maintainers directly at: xuhuo4149@gmail.com.
3. Provide as much detail as possible (steps to reproduce, affected code, potential impact).
4. We will acknowledge receipt within 72 hours and work on a fix promptly.

## Disclosure Policy
- We aim to fix confirmed vulnerabilities within a reasonable timeframe.
- Once a fix is available, we will publish a new release and update this document.
- Credit will be given to reporters unless anonymity is requested.
