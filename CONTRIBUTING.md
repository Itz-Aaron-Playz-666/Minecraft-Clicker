# Contributing to Minecraft Clicker Game

We welcome contributions! By participating in this project, you agree to follow our code of conduct and licensing terms (GPLv3).

## How to Contribute

### Reporting Issues
- Use the GitHub Issues tab.
- Clearly describe the problem, steps to reproduce, and expected behavior.
- Include screenshots or logs if relevant.

### Submitting Changes
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/your-feature-name`).
3. Make your changes.
4. Run tests and ensure the build passes (`npm run build`).
5. Commit with clear messages (`feat: add click multiplier`).
6. Push to your fork and open a Pull Request.

### Coding Standards
- Use **TypeScript** for all source code.
- Follow ESLint + Prettier rules (configured in `.eslintrc` and `.prettierrc`).
- Keep functions small and focused.
- Write meaningful commit messages.

### Development Setup
- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Run tests: `npm test`

### Code of Conduct
Please be respectful and constructive. Harassment or abusive behavior will not be tolerated.

Thank you for helping improve the Minecraft Clicker Game!
