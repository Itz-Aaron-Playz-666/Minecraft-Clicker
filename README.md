# Minecraft Clicker Game

A fun, open-source clicker game inspired by Minecraft mechanics.  
Built with **TypeScript** and powered by **Vite.js** for fast development and cross-platform support.

---

## 🚀 Features
- Simple clicker mechanics (press to earn points).
- Minecraft-inspired visuals and themes.
- Cross-platform: works on web, desktop, and mobile.
- TypeScript for type safety and maintainability.
- Vite.js for blazing-fast builds and hot reload.

---

## 📦 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/minecraft-clicker-game.git
cd minecraft-clicker-game
pnpm install
```
🛠 Development
Start the Vite dev server:
```bash
pnpm run dev
```
Build for production:
```bash
pnpm run build
```
Preview the production build:
```bash
pnpm run preview
```
